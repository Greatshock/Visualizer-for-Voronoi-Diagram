Voronoi Diagram Visualizer (Fortune Algorithm)

Manual

Author: Nikita Marinosyan

Introduction

This application is developed as a coursework assignment at HSE Faculty of Computer Science.
The main purpose is to illustrate Voronoi diagram building allowing user to watch and control
the construction of the diagram step by step.

User interface

User interface consists of:
  • Canvas where the diagram building is shown
  • Control panel below the canvas
  • Control panel above the canvas
  • Menu panel

Canvas

Key features:
  • Click on the canvas to place the site(point)
  • Adding sites also allowed while building the diagram

Control panel (Top)

Contains checkboxes which allows users to choose which events and parts of the diagram they want to see.
  • Circle events – reflects whether the circle events are shown
  • Beachline - reflects whether the beachline is shown
  • Diagram – reflects whether the completed part of diagram is shown
  • Legend – gives the information about what lines of different colors mean

Control panel (Bottom)

Contains buttons which controls the process of building the diagram.
  • Pause – interrupts the diagram building
  • Start/Resume – starts/Resumes the diagram building
  • Next step – steps 1 pixel forward
  • Next event – moves to the next event of Fortune algorithm
  • Restart – starts building the diagram again
  • Clear – clears the canvas

Menu

Main features:
  • Help
    • Manual
      • - opens the manual (You are reading it now)
    • About the program
      • - pops up the window with brief info about the application
  • Open web version – opens the app in browser
